
# Natapp 快速上手指南（国内）

## 快速开始

### 1. 注册账号

访问 https://natapp.cn/ 注册

### 2. 购买免费隧道

1. 登录后点击「购买隧道」
2. 选择「免费隧道」
3. 配置：
   - 本地端口：`3900`
   - 协议：`Web`

### 3. 下载客户端

下载 Windows 版本 natapp

### 4. 启动 Vibe Bridge

```bash
npm run dev:server
```

### 5. 运行 natapp

```bash
natapp -authtoken=你的隧道token
```

### 6. 手机访问

natapp 会显示分配的域名，手机打开即可！

## 注意事项

- 免费版带宽 1Mbps，测试够用
- 国内服务器，速度快
- 适合国内测试使用
